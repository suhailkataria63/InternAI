# InternAI API Reference

## Base URL

Local development:

```text
http://127.0.0.1:8000
```

## Health Check

### `GET /health`

Returns the current backend health status.

Example request:

```bash
curl http://127.0.0.1:8000/health
```

Example response:

```json
{
  "status": "ok",
  "project": "InternAI",
  "message": "Backend is running"
}
```

Status code: `200 OK`

## Resume Upload

### `POST /api/resume/upload`

Accepts a resume PDF upload, extracts raw text with PyMuPDF, and returns the extracted text.

The request must use `multipart/form-data` with a `file` field.

Example request:

```bash
curl -X POST http://127.0.0.1:8000/api/resume/upload \
  -F "file=@/path/to/resume.pdf;type=application/pdf"
```

Example response:

```json
{
  "filename": "resume.pdf",
  "text_length": 1234,
  "extracted_text": "..."
}
```

Validation error for non-PDF uploads:

```json
{
  "detail": "Only PDF resume uploads are supported."
}
```

Success status code: `200 OK`

Validation error status code: `400 Bad Request`

## Resume Analysis

### `POST /api/resume/analyze`

Accepts raw resume text and returns a structured profile generated by the Resume Analyzer Agent. This endpoint works without an external LLM API key.

Example request:

```bash
curl -X POST http://127.0.0.1:8000/api/resume/analyze \
  -H "Content-Type: application/json" \
  -d '{
    "resume_text": "Jane Doe\njane@example.com\nSkills\nPython, FastAPI\nProjects\nInternship tracker app"
  }'
```

Example response:

```json
{
  "profile": {
    "name": "Jane Doe",
    "email": "jane@example.com",
    "phone": "",
    "education": [],
    "skills": ["Python", "FastAPI"],
    "projects": ["Internship tracker app"],
    "experience": [],
    "certifications": [],
    "strengths": [
      "Includes a clear skills section.",
      "Highlights project experience."
    ],
    "improvement_areas": [
      "Add a phone number for recruiter contact.",
      "Add internship, work, volunteer, or leadership experience."
    ]
  }
}
```

Success status code: `200 OK`

Validation error status code: `422 Unprocessable Entity`

## Match Scoring

### `POST /api/match/score`

Compares a parsed resume profile with a parsed job profile and returns a deterministic internship fit score. This endpoint works without an external LLM API key.

Scoring formula:

- Required skill match: 50 points
- Preferred skill match: 15 points
- Project relevance: 20 points
- Education and role relevance: 10 points
- Experience and certifications: 5 points

Example request:

```bash
curl -X POST http://127.0.0.1:8000/api/match/score \
  -H "Content-Type: application/json" \
  -d '{
    "resume_profile": {
      "name": "Jane Doe",
      "email": "jane@example.com",
      "skills": ["Python", "FastAPI", "React"],
      "projects": ["Built a FastAPI internship tracker with SQL database"],
      "education": ["B.S. Computer Science"],
      "experience": [],
      "certifications": []
    },
    "job_profile": {
      "role_title": "Software Engineering Intern",
      "required_skills": ["Python", "FastAPI", "SQL"],
      "preferred_skills": ["React"],
      "responsibilities": ["Build backend APIs", "Write SQL queries"],
      "eligibility": ["Computer Science students preferred"],
      "keywords": ["Python", "FastAPI", "SQL", "Backend"]
    }
  }'
```

Example response:

```json
{
  "match_score": 78,
  "match_level": "Strong Fit",
  "matched_skills": ["Python", "FastAPI", "React"],
  "missing_skills": ["SQL"],
  "project_relevance_notes": [
    "Project 'Built a FastAPI internship tracker with SQL database' matches job terms: FastAPI, SQL, Intern."
  ],
  "recommendation": "Improve the match by adding evidence for these required skills: SQL."
}
```

Success status code: `200 OK`

Validation error status code: `422 Unprocessable Entity`

## Job Description Analysis

### `POST /api/jobs/analyze`

Accepts pasted internship or job description text and returns a structured job profile generated by the JD Analyzer Agent. This endpoint works without an external LLM API key.

Example request:

```bash
curl -X POST http://127.0.0.1:8000/api/jobs/analyze \
  -H "Content-Type: application/json" \
  -d '{
    "job_description": "Role: Software Engineering Intern\nCompany: Acme Labs\nLocation: Bengaluru\nWork mode: Hybrid\nDuration: 6 months\nStipend: INR 25000 per month\nRequired Skills\nPython, FastAPI, SQL\nResponsibilities\nBuild backend APIs"
  }'
```

Example response:

```json
{
  "job_profile": {
    "role_title": "Software Engineering Intern",
    "company_name": "Acme Labs",
    "required_skills": ["Python", "SQL", "FastAPI"],
    "preferred_skills": [],
    "responsibilities": ["Build backend APIs"],
    "eligibility": [],
    "stipend": "INR 25000 per month",
    "duration": "6 months",
    "location": "Bengaluru",
    "work_mode": "Hybrid",
    "keywords": [
      "Python",
      "SQL",
      "FastAPI",
      "Software Engineering Intern",
      "Acme Labs",
      "Bengaluru",
      "Hybrid"
    ]
  }
}
```

Success status code: `200 OK`

Validation error status code: `422 Unprocessable Entity`
